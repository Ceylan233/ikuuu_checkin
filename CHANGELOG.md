# v1.0.1

- 自定义 SMTP 字段调整到发件邮箱类型选择框正下方。
- 使用标签页分离账号、验证码和邮件设置，避免依赖字段错位。
- 修复空收件人列表导致包装器提前退出的问题。
- 软件包升级后按启用状态自动恢复定时服务。

## v1.0.0

- 支持 163、126、QQ/Foxmail、Gmail、Yahoo、Outlook 和自定义 SMTP。
- 支持一个发件人和多个收件人，自定义 SMTP 可单独填写登录用户名。
- 新增 OpenWrt/iStoreOS LuCI 配置页、每日定时、立即签到、测试邮件和日志。
- 提供可直接使用 `opkg install` 安装的 IPK。
